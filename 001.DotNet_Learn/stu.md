## .net生态
.net有多种实现: 因为历史原因 和  技术原因

- .NET Framework : 最初的.net,仅提供了windows平台运行能力
- Mono : 原始社区和开源.net,跨平台实现的.NET Framework
- .NET (Core)  : 现在.net ,跨平台和开源实现，积极支持 Linux , Macos , Windows
